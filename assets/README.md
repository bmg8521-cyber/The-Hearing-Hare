## Asset Guidelines

### Logo and Branding
- **Logo**: Should feature a hare/rabbit character that represents listening and mental health
- **Color Scheme**: Maintain calming colors (blues, purples, soft tones)
- **Mascot**: Friendly, approachable character that embodies empathy and music

### Images
- **Format**: Use modern formats (WebP, SVG when possible)
- **Accessibility**: Include meaningful alt text for all images
- **Performance**: Optimize all images for web (under 500KB for photos)
- **Responsive**: Provide multiple sizes for different screen densities

### Icons
- **Style**: Consistent with overall design (rounded, friendly)
- **Format**: SVG preferred for scalability
- **Accessibility**: Proper ARIA labels and descriptions

## Content Notes

All assets are currently placeholders and should be replaced with:
